Difference between graphql and rest api
1. GraphQl has single endpoint whereas rest api has multiple endpoints.
2. Graphql works on server whereas rest api works on database..
3. Graphql gives that data only which is required or requested by client through server response..
4.REstFul api has problem of overfetching and underfetching which need more bandwidth.
